Structure

IMPORTANT: Two versions of the pack exist:

  • The ITG version: Has +0.009ms offsets applied, has 4:3 (800x600) resolution background art
  • The “SM5” version: Has no changes to offsets applied, has 16:9 (1920x1080) resolution background art
    • It is a misnomer to call it the “SM5 version" because StepMania 5 users still use ITG's mechanics. This would be called the “null offset” version in 2024

Trivia

  • Dragonforce Kaioken and Cirque du Miura were released alongside an upgraded and optimized simfile release spreadsheet. I managed to reduce the number of spreadsheets per pack used from 3 to 2, whilst maintaining the quantity of information shared
    • The stats sheet now showed every chart next to each other, and was set up so that alphabetical ordering would display all the information in correct order to allow sorting of charts for set planning
